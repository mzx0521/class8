# 图片资源文件夹

此文件夹用于存放作品展示页面的所有图片资源。

## 图片文件命名规范

### 项目路演模块
- `roadshow-pain-point.png` - 痛点场景图（会议场景）
- `roadshow-plugin-install.png` - 插件安装与启动展示
- `roadshow-ai-advice.png` - 实时军师建议展示
- `roadshow-download-docx.jpg` - 一键生成纪要展示

### 其他作品模块
- `work-greeting-card.jpg` - 新春贺词生成器截图
- `work-line-art.jpg` - 图片拼接线稿生成器截图
- `work-prd-extractor.jpg` - 需求提取 Skills 截图
- `work-personal-home.jpg` - 个人主页截图

## 如何替换图片

1. 将您的图片文件放入此文件夹
2. 确保文件名与上述命名规范一致
3. 支持的格式：.png, .jpg, .jpeg, .webp
4. 建议图片尺寸：
   - 项目路演图片：1200x800 或 16:9 比例
   - 作品截图：1200x900 或 4:3 比例

## 注意事项

- 图片文件名区分大小写
- 建议压缩图片以提升加载速度
- 如需使用其他文件名，请修改 `src/constants/images.ts` 配置文件
